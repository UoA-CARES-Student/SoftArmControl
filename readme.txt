# Readme
The chatbot is in Chatbot folder and the vision control system is in yolov5 folder

Note: Ensure a intel realsense D435 or D435i is plugged in, and a Nvidia GPU with at least 4gb of vram is also present


## Usage Instructions

Running main.py will start the whole system.
To stop the system, press q in the vision control system live view window, and close the chatbot window. 
Pressing ctrl+c in the console may also be required

```
testyolo.py 
```

just tests the vision control system without running the chatbot
